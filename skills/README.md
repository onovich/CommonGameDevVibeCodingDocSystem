# Skills 目录说明

本目录包含可复用的 skill 模板。

## Skill 设计原则

参考 `workflow/thin-skill-heavy-doc.md`。

## 推荐 Skill 模板

### 通用 Skill

- `low-churn-refactor` - 低 churn 重构调度
- `verify-compile` - 编译验证
- `phased-feature` - 阶段化功能开发

### Unity Skill

- `unity-refactor-playbook` - Unity 原型收敛
- `unity-batchmode-verify` - Unity 无头编译验证

## 使用方式

将 skill 模板复制到项目的 `.claude/skills/` 目录，根据项目调整。
